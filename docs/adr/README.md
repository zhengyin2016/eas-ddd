## ADR模板

每个ADR包含以下部分：

- **状态**: 提议中/已接受/已废弃/已替代
- **上下文**: 决策背景和问题描述
- **决策**: 具体决策内容
- **理由**: 为什么做出这个决策
- **后果**: 决策带来的正面和负面影响
- **替代方案**: 考虑过但未采纳的方案

## 如何新增ADR

1. 复制模板创建新文件：adr-XXX-title.md
2. 按模板填写内容
3. 更新本README索引
4. 提交PR评审
